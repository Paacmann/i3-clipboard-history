# i3-clipboard-history

A small, useful clipboard history tool for i3 window manager users , written in C++

## Features
- Saves the last N copied clipboard entries
- Allowes quick selection using **dmenu** or **rofi**
- Lightweight and minimal 

## Technologies
- **C++** 
- **xclip (clipboard interaction)**
- **dmenu or rofi (menu selection)**
- **i3 window manager**
